Install `tor` using

```bash
sudo aptitude install tor
```

For GUI, install [torbrowser-launcher](https://github.com/torproject/torbrowser-launcher)

```bash
flatpak install flathub org.torproject.torbrowser-launcher
```

To find the ip address,

```bash
nmcli device show
```

```bash
hostname -I
```

```bash
ip addr show
```
